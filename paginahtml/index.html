Estrutura básica HTML

A estrutura básica do HTML é onde tudo começa, é preciso de uma estrutura para definir as tags dentro do seu código, primeiro definismo a tag HTML, dentro da tag HTML definismo a tag head e fora da tag head colocamos a tag body, a tag head significa "cabeça" é o que vai ser inserido no cabeçalho do site em contra partida a tag body é o que vai mostrar no corpo do site.

Tags em html

As tags html são a formatação de um algum tipo de um conteúdo que existe o title que é inserido dentro do head para dar nome à página, o H1 são para títulos dentro do corpo a tag strong deixa o texto em negrito, a tag i deixa em itálico, a tag input gera um campo para o usuário.

Atributos das Tags

As tags podem ter uma configuração específico e chamamos isso de atributos, na tag input existe o tipo de texto, número e de cor e também podemos ter as classes.

Tipografia

Temos como formatamos os textos em HTML da maneira que quisermos, existe algumas Tags, reponsáveis por isso, como o H1,H2,H3 e etc que define o título, sub-título, sub-título do sub-título e etc e também temos as opções de customização dos textos que temos o mark que deixa marcado o nome e como já falado o strong que é negrito, a tag i que é itálico e a tag u que sublinha o texto e também o html possui a tag blockquote que é responsável for formatar citações.

Listas Ordenadas e Não Ordenadas

Existem 2 tipos de listas as ordenas e não ordenas, as ordenas usam a tag ol e ela enumera cada item de acordo com o número inicial 1 , em contra partida a tag não ordenada é a tag ul que não possui ordem.

Links

As páginas webs possuem links, e no HTML possui uma tag específico de redirecionamento de links que é a tag a, com está tag podemos inserir links ou outros arquivos htmls e o navegador irá redirecionar para o link escolhido pelo o usuário
